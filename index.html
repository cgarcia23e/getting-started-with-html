The Basics of the Web and HTML
The worldwide web is made up of HTML documents that are processed and communicated via computers. The HTML documents are what glue evertything together and form the basis of the web. Hyperlinks link everything; making the web weblike.

The way the process starts is, an individual goes to a webpage by sending an HTTP request to a server via a computer. The server then searches for the appropriate html document and sends it back to the user's page where the web browser interprets the information into a form that can be seen on the user's screen.

HTML (Hyper Text Markup Language)
As mentioned before, these documents make up the heart of the web. HTML consists of:
-Text context (what you see)
-Markup (what the information will look like)
-References to other documents (i.e. images and videos)
-Links to other pages

HTML Markup: are made up of tags using the less than and greater than symbols and contribute to the aesthetics and functionality of content. There are multiple uses for markups. For the most part, HTML markups contain an opening tag , the content is then written after the opening tag, and is followed by a closing tag. This whole expression is called an element. You can put content in bold, italics; create links for additional content; and divide sentence structures. You can also nest multiple markups with each other.

That said, because computers do not have the flexibility as humans, the commands and element markups must be precise; otherwise mistakes can produce huge programming issues. Being familiar with whitespace and inline vs. block elements can help demonstrate how the markup language must be literal.

Inline vs. Block Elements
White space consists of the characters that represent horizontal or vertical space in typography. HTML elements are all found in the white space and because the typography is represented in either a horizontal or vertical space, HTML elements are either inline ( in the same horizontal thred) or block ( can shift and stay within it's own vertical block space).
